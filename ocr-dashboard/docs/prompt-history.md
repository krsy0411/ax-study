# 프롬프트 입력 히스토리

아래는 이번 작업에서 핵심 프롬프트를 요약한 기록입니다.

## 1) 초기 구현 요청

- 프롬프트 요지
  - PRD를 참고해 HTML 파일 생성
  - OCR 인식률 강화
  - 영수증(표 구조) 특성 고려
  - 전처리/정확도 향상 라이브러리 적극 사용
- 결과
  - [index.html](../index.html) 기반 OCR 대시보드 1차 구현

## 2) UX 변경 요청 (사전 컨펌)

- 프롬프트 요지
  - 같은 키워드에 값이 2개 이상 나오는 케이스 대응
  - 후보값을 리스트업하고 사용자 선택 가능하도록 변경
  - 작업 전에 이해 여부 확인 요청
- 결과
  - 요구사항 컨펌 후 후보 선택 UX 구현

## 3) 엑셀 처리 방식 변경 요청

- 프롬프트 요지
  - 다운로드 방식이 아니라 기존 엑셀 선택 후 새 시트에 업데이트
  - 열 너비 자동 조정
- 결과
  - 기존 파일 읽기 + 새 시트 append + auto-fit 반영

## 4) 덮어쓰기 가능 여부 질의

- 프롬프트 요지
  - 선택한 엑셀 파일 원본 덮어쓰기 가능 여부
- 결과
  - 브라우저 보안 제약 안내
  - 기본 동작은 새 파일 저장 방식으로 유지
