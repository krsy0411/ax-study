# 핵심 질문에 답해보기

비개발자들이 할만한 질무들

## 엑셀 내 공통 질문
1. **OCR이 뭔가요?**

   이미지 속 글자를 텍스트로 변환하는 기술. 여행갔을 때, 일본어 간판을 사진 찍으면 그 글자를 인식해서 한국어로 변환해주는 것도 OCR이라는 텍스트 변환 기술을 이용한 것.
2. **PDF에서 데이터를 뽑는게 가능한가요?**
   
    PDF는 두 가지 종류가 존재 :
    | 종류 | 설명 | 결론 |
    | -- | -- | -- |
    | **텍스트 PDF** | 워드 같은 문서에서 변환한 PDF | OCR 기술을 사용하지 않아도 곧바로 추출 가능 |
    | **이미지 PDF** | 스캔한 서류(스캔본) | OCR 기술을 사용해야 텍스트 추출 가능 |

    컴퓨터 인식 가능한 글자인 "유니코드(Unicode)"라는게 존재하고, 이건 이미 컴퓨터가 인식 가능하기에 텍스트 PDF 같은 경우엔 곧바로 추출 가능. 
    
    반면 이미지 PDF의 경우엔 컴퓨터가 인식 가능한 글자가 아니라 이미지를 구성하는 픽셀로 인식함. 예를 들어, "진료비 15,000원"은 수백만 개의 픽셀(RGB 값)으로 존재할 뿐임.

    그래서! OCR 기술은 픽셀 패턴을 분석해서 컴퓨터가 알고 있는 문자 코드(Unicode)로 변환하는 기술임.

    ```mermaid
    graph LR
        A[픽셀 배열 - 그림] --> B[윤곽선 감지 - 전처리]
        B --> C[패턴 매칭 - 딥러닝 ]
        C --> D[Unicode 변환 - 텍스트]
    ```
3. **저는 OCR같은 전문 용어를 모르는데, 어떻게 GPT나 Copilot한테 요청해야해요?**

   예전에는 "OCR로 PDF 파싱해줘"라고 해야 AI가 OCR이라는 기술을 이용해 PDF에서 텍스트를 추출할 수 있도록 코드를 작성했는데, 이제는 그냥 "영수증 사진에서 금액 숫자들만 뽑아 목록으로 만들고 싶어"라고만 해도 OCR 기술을 사용함.
4. **PDF에서의 데이터 추출은 되는데, 바로 데이터를 분석해서 엑셀에 저장하는 것도 가능한가요?**
   
    "PDF 추출 -> 분석 -> 엑셀 저장"은 "PDF 또는 이미지 업로드 -> OCR로 텍스트 추출 -> 엑셀(.xlsx) 저장" 흐름으로 코드를 작성하면 구현 가능함.
