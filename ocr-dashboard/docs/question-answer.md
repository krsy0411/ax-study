# 핵심 질문에 답해보기

비개발자들이 할만한 질무들

## 엑셀 내 공통 질문
1. OCR이 뭔가요?
2. PDF에서 데이터를 뽑는게 가능한가요?
3. 저는 OCR같은 전문 용어를 모르는데, 어떻게 GPT나 Copilot한테 요청해야해요?
4. PDF에서의 데이터 추출은 되는데, 바로 데이터를 분석해서 엑셀에 저장하는 것도 가능한가요?

## 추가 예상 질문

- OCR 결과가 잘못 나오면 어떻게 수정하나요?
- 엑셀로 저장된 결과를 다시 불러와서 이어서 작업할 수 있나요?
- 여러 장의 영수증을 한 번에 처리할 수 있나요?
- 이미지가 흐릿하거나 구겨진 경우에도 인식이 잘 되나요?
- 개인정보(이름, 주민번호 등)는 자동으로 가려지나요?
- 엑셀 시트에 저장된 데이터 형식을 바꿀 수 있나요?
- 프로그램 설치 없이 바로 쓸 수 있나요?
- 모바일에서도 사용할 수 있나요?
- 결과를 이메일이나 다른 사람과 쉽게 공유할 수 있나요?
- OCR 처리 속도가 느릴 때는 어떻게 하나요?
- 지원하는 파일 형식(PDF, JPG, PNG) 외에 다른 형식도 쓸 수 있나요?
- 추출된 데이터에 오타가 있으면 직접 고칠 수 있나요?
- 엑셀 파일이 너무 커지면 문제가 생기나요?
