# 피드백 정리

## 피드백 1: 영수증 OCR 정확도를 높이고 싶다

- 요청 요지
  - 칸이 많은 영수증 이미지 특성상 OCR 인식률을 최대한 높이고 싶음
  - 이미지 전처리/정확도 향상 라이브러리 적극 활용 요청
- 반영 내용
  - OpenCV.js 전처리(적응형 이진화, morphology)
  - 고정밀 모드(업샘플링 + 다중 OCR 패스)
  - 라인 병합 및 키워드 매칭 보강

## 피드백 2: 키워드당 값이 2개 이상인 경우가 많다

- 요청 요지
  - 자동 단일 선택보다 후보를 모두 보여주고 사용자가 선택하도록 UX 개선
- 반영 내용
  - 키워드별 숫자 후보 목록 생성
  - 후보 드롭다운 선택 UI 추가
  - 선택값 기준으로 결과 테이블 및 엑셀 반영

## 피드백 3: 엑셀은 다운로드가 아니라 기존 파일 업데이트 형태

- 요청 요지
  - 사용자가 엑셀 파일을 고르면 새 시트를 생성해 결과를 추가하는 형태 필요
  - 셀 텍스트 길이에 맞춘 열 너비 자동 조정 필요
- 반영 내용
  - 기존 엑셀 파일 선택 입력 추가
  - 선택 파일을 읽어 새 시트 append
  - 열 너비 auto-fit 적용
  - 시트명 중복 시 자동으로 유니크 처리

## 피드백 4: 선택한 엑셀 파일 덮어쓰기 가능 여부

- 결론
  - 순수 로컬 HTML + 일반 file input 모델에서는 브라우저 보안 제약으로 원본 덮어쓰기는 일반적으로 불가
  - 대안으로 업데이트 결과를 새 파일로 저장
