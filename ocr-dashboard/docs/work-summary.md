# 작업 정리

## 1) 초기 구현 (PRD 기반)

- PRD([product-requirement-document.md](../product-requirement-document.md))를 기준으로 단일 HTML 대시보드 구현
- 메인 구현 파일: [index.html](../index.html)
- 주요 기능
  - PDF/JPG/PNG 업로드
  - OCR 추출(Tesseract.js)
  - 전처리 강화(OpenCV.js + 기본 대비 보정)
  - PDF 페이지 렌더링(pdf.js)
  - 결과 테이블 표시 및 엑셀 저장(SheetJS)

## 2) OCR 정확도 강화 작업

- 고정밀 모드 제공
  - 스케일 업 샘플링
  - 다중 OCR 패스(psm 6 + psm 11)
  - 라인 병합으로 텍스트 후보 보강
- 영수증 표 구조 특성을 반영한 숫자 추출 로직 적용

## 3) UX 개선: 키워드별 다중 후보 선택

- 기존: 키워드당 단일 값 자동 선택
- 개선: 키워드당 숫자 후보 리스트업 후 사용자가 직접 선택
- 구현 내용
  - 키워드 유사도 + OCR 신뢰도로 후보 점수화
  - 후보 중복 제거 및 상위 후보 노출
  - 사용자 선택 즉시 결과 테이블 동기화

## 4) 엑셀 처리 방식 변경

- 기존: 새로운 엑셀 파일 다운로드
- 개선: 사용자가 선택한 기존 엑셀 파일을 읽고 새 시트를 추가한 뒤 저장
- 추가 적용
  - 시트명 충돌 방지(자동 suffix)
  - 셀 텍스트 길이에 기반한 열 너비 자동 조정

## 5) 현재 상태

- 핵심 기능은 [index.html](../index.html)에 통합
- 산출물 관련 문서는 [README.md](../README.md) 및 [docs](.) 하위 파일로 정리
