# OCR 대시보드(제품 요구사항 정의서)

## 1. 개요
이 프로젝트는 로컬 HTML 파일만 사용해 AX 교육 참여자들이 의료비 서류(PDF,JPG, PNG)를 업로드하면 OCR를 통해 텍스트를 추출하고, 진료과목,검사료,영산진단료, CT 진단료 정보를 추출해 엑셀에 정리하는 웹 애플리케이션입니다.

## 2. 기능
- 파일 업로드
- AI OCR를 통한 텍스트 추출 : 진료과목,검사료,영산진단료, CT 진단료 정보 추출
- 엑셀에 정리

## 3. 기술 스택
- HTML
- CSS
- JavaScript(HTML 문서에서 사용. 별도 파일 분리는 하지 않음)
- OCR : Tesseract.js
- 엑셀 : SheetJS

## 사용자 흐름
- 사용자가 업로드 버튼을 클릭해 파일을 업로드(파일 선택)
- 업로드된 파일이 화면에 표시
- 영수증 내 여러 값들 중 어떤 값(예: 진료과목, 검사료, 영상진단료, CT 진단료)을 추출할지 사용자가 텍스트(쉼표로 구분)로 입력
- 추출 버튼 클릭
- OCR을 통해 텍스트 추출 진행
- 사용자가 입력한 값들과 매핑해 엑셀에 정리(파일 선택) : 이때, 매 생성마다 엑셀 파일 내 시트들을 분리(날짜별 처리)
